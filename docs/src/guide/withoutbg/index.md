# withoutbg

::: warning withoutbg について

- dev-m1sk9-s1 での提供は現在試験段階です．
- dev-m1sk9-s1 は CPU があまり良い物ではないことから，画像透過には約10秒ほどかかります．
    - 大量の画像を透過したい場合は別サービスを使ってください．
- withoutbg は AI モデルです．あなたがアップロードした画像がモデル学習に使われる可能性があります．責任は負いません．自己責任で使ってください．
    - 一応公式では [**使用しない**](https://withoutbg.com/#:~:text=Do%20you%20use%20customer%20data) と明言はされています．

:::

withoutbg は画像の背景を除去する AI モデルです．

## 使い方

::: danger 日本国外からのアクセスについて

不正利用防止のため，日本国外からのアクセスは禁止にしています．

また VPN を使用しているとアクセスできなくなるので注意してください．

:::

1. [withoutbg-s1.m1sk9.dev](https://withoutbg-s1.m1sk9.dev) にアクセスする
2. メール認証を通過する
    - 自分が使用しているメールアドレスを入力して，返信される確認コードを入力してください
    - メールアドレスの制限はありません．不正利用防止のための処置です．

![](/media/cloudflare-zt-01.png)

3. 画像をドラッグまたはクリックして選択する

![](/media/withoutbg-01.gif)

4. 数十秒待つ
5. 透過加工された画像が出てくるのでダウンロードする

![](/media/withoutbg-02.png)

## ルール

- 日本国内からの利用に限ります．
- 公序良俗に反する画像のアップロードは禁止します．
- 不正利用が発覚した場合，利用を停止することがあります．
- 利用に関して発生した損害について，管理者は一切の責任を負いません．
- その他，管理者が不適切と判断した場合，利用を停止することがあります．
